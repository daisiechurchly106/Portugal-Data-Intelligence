## 🚀 What this app does

Portugal-Data-Intelligence is a desktop-ready analytics project for the Portuguese economy from 2010 to 2025. It brings data into one place, checks it for errors, stores it in SQL, and prepares reports you can use in Power BI or review with AI insights.

Use it to:

- Review key macroeconomic trends
- Work with clean, structured data
- Create charts and reports
- Track changes over time
- Compare economic indicators in one place

## 📊 Main features

### 1. ETL pipeline

The app pulls in economic data, cleans it, and prepares it for use. This helps keep the data in one format so it is easier to review.

### 2. Data quality checks

It checks for missing values, wrong dates, duplicates, and bad records. This helps keep the output clean.

### 3. SQL database storage

The project stores processed data in SQLite or SQL-based tables. This makes it easier to query and reuse the same data.

### 4. Power BI reporting

You can use the prepared tables in Power BI Desktop to build dashboards for GDP, inflation, unemployment, trade, and other indicators.

### 5. AI insights

The project can prepare short written insights from the data. These help turn tables and charts into plain-language notes.

### 6. Forecasting support

It may include forecasting files for future trends based on past data. This helps with planning and comparison.

## 🧰 Troubleshooting

### The ZIP file will not open

- Download it again
- Make sure the file finished downloading
- Try extracting it with Windows Extract All

### The script does not run

- Check that Python is installed
- Open the terminal from the project folder
- Run the correct main file
- Make sure you installed the required packages

### Power BI does not load the data

- Check that the source files are still in the same folder
- Refresh the data connection
- Confirm that the database or CSV files still exist

### The database file is empty

- Run the ETL step again
- Check for errors in the script
- Make sure the input data files are present
